### Start the local web server

Execute the npm script **clean** followed by **start** in the website sub-project,
or execute the following within the project folder:

```sh
npm run clean -C website
npm run start -C website
```

Navigate to the **Maintainer Info** page,
the **[Start the local web server](http://localhost:3000/doxygen2docusaurus-cli-ts/docs/maintainer/#start-the-local-web-server)** section.
