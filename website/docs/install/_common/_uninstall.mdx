### Uninstall

The xpm packages do not use any form of system installer; instead they
are distributed as
portable archives;
therefore they do not require to run any uninstaller; simply removing the
links and possibly the user's global xPacks store folder and
the user xPack cache folder is enough.

To remove the links created by
**xpm** in the current project,
go to the project folder:

```sh
cd my-project
```

and ask **xpm** to uninstall the package:

<CodeBlock language="sh"> {
`xpm uninstall @xpack/doxygen2docusaurus --verbose
`} </CodeBlock>

To completely remove the package from the user's global xPacks store:

<CodeBlock language="sh"> {
`xpm uninstall --global @xpack/doxygen2docusaurus --verbose
`} </CodeBlock>
